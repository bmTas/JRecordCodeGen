Standard templates

The standard template produces 3 java class files
  • FieldName class which holds the cobol field names as constants
  • Skeleton Read program. This is a basic Read ~ Print program. For Multi-Record files you will need to add Record-Selection code.
  • Skeleton Write program.

In this directory there are the following generate scripts

  • genDTAR020.bat / sh - Generate JRecord code for a single record Mainframe EBCDIC fixed-width file
  • genAmsPODownload.bat / sh - Generate JRecord code for a Multi-Record (with arrays) Ascii Text file

To produce skeleton programs using Single record copybook (taken from genDTAR020.bat):


  java -jar ../../lib/JRecordCodeGen.jar  ^
               -Template           standard ^
               -package            example.ioBuilder.dtar020 ^ 
               -Schema             DTAR020.cbl   ^
                 -FileOrganisation FixedWidth ^
                 -font cp037 ^
                 -DropCopybookName   true ^
               -outputDirectory    DTAR020
