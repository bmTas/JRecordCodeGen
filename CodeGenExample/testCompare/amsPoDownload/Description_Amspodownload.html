IPoRecordPojo Interface

The IPoRecordPojo Interface has get.../set... methods for all the Cobol fields in the PoRecord Record Definition. The Interface IPoRecordPojo is implemented by 2 classes:

  • LinePoRecordJR.java is Wrapper for a standard JRecord Line with the appropriate Getter/setter methods. Behind the scenes it call the line.getFieldValue(..) ... methods to get/retrieve the values. There are the 2 other Methods for converting to/from a LinePoRecordPojo:
    • toPojo() method which converts this line into a LinePoRecordPojo class.
    • set(IPoRecord value) which will update the line with the values from a LinePoRecordPojo line.
  • LinePoRecordPojo.java implments the Interface using standard java types. It does not use or reference JRecord.

The IPoRecordPojo Interface extends 2 other interfaces:

  • The IPoRecord Interface defines the get... methods.
  • The IPoRecordUpd Interface defines the set... methods.

Finally there is the Code.java class that has methods to copy between to classes that implement the IPoRecordPojo interface.
