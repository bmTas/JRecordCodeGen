Other Templates

This directory holds bat/shell script programs to generate Java Programs using the schemaClass and javaPojo templates. Just try them and check out the code. The Bat shell scripts available are:

  • genDTAR020.bat / sh generate java code for the DTAR020.cbl copybook using the schemaClass and javaPojo templates
  • genAmsPODownload.bat / sh generate java code for the amsPODownload.cbl copybook using the schemaClass template

For the initial release the focu was on the standard template
