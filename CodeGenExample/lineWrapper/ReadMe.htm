linewrapper template

The linewrapper template produces at least 4 java class files
  • FieldName class which holds the cobol field names as constants
  • LineWrapper class for each record type. A LineWrapper is a 'wrapper' for a JRecord line providing get.../set... methods for the Cobol fields.
  • Skeleton Read program. This is a basic Read ~ Print program. For Multi-Record files you will need to add Record-Selection code.
  • Skeleton Write program.

In this directory there are the following generate scripts

  • genDTAR027.bat / sh - Generate JRecord code for a single record Mainframe EBCDIC fixed-width file
  • genAmsPODownload.bat / sh - Generate JRecord code for a Multi-Record (with arrays) Ascii Text file

To produce skeleton programs using Single record copybook (taken from genDTAR027.bat):


  java -jar ../../lib/JRecordCodeGen.jar  ^
               -Template           lineWrapper ^
               -package            lineWrapper.dtar027 ^ 
               -Schema             DTAR027.cbl   ^
                 -FileOrganisation FixedWidth ^
                 -font cp037 ^
                 -DropCopybookName   true ^
               -outputDirectory    DTAR027
